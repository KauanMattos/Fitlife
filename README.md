# 💪 FitLife

FitLife é um projeto de site fitness responsivo desenvolvido com HTML, CSS e JavaScript, com foco em design moderno, experiência do usuário e adaptação para diferentes dispositivos.

## 📸 Preview
![Preview do projeto](./readme-images/desktop.png)

---

## 🚀 Sobre o projeto

O FitLife foi criado com o objetivo de simular uma plataforma voltada ao mundo fitness, apresentando serviços, planos e incentivo a um estilo de vida saudável.

O projeto explora conceitos importantes de desenvolvimento front-end, como:

- Estruturação semântica com HTML5
- Estilização avançada com CSS3
- Interatividade com JavaScript
- Responsividade (mobile, tablet e desktop)
- UI/UX moderna

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)

---

## 💻 Funcionalidades

- Layout totalmente responsivo
- Seções de apresentação de serviços
- Interface moderna e intuitiva
- Navegação fluida
- Design focado em experiência do usuário
